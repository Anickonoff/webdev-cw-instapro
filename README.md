# Instapro

MVP социальной сети для обмена фотографиями

## Первоначальная оценка

Подготовительные работы 0,5 часа
Основные работы 8 часов
Дополнительные работы 2 часа
Список запланированных дополнительных работ:

* Функционал удаления постов
  * Возможность удаления только своих постов
* Список людей, поставивший лайк, в отдельном popup окне

## Фактически затраченное время

Подготовительные работы 0,7 часа:

* установка и настройка dev модулей (prettier, eslint и т.п.)

Основные работы 4,3 часа:

* интеграция вёрстки постов с API
* страница добавления поста
* страница постов отдельного пользователя
* функционал лайков
* добавление модуля date-fns для отображения даты поста
* адаптивный дизайн для экранов уже 500px
* фильтр тегов как при отображении страницы постов, так и при отправке нового поста или регистрации пользователя

Дополнительные работы 1,5 часа
Сделано:

* Функционал удаления постов по нажатию на кнопку корзины
  * Возможность удаления только своих постов
* Список людей, поставивший лайк, пишется вместо числа лайков
