## Первоначальная оценка

Подготовительные работы (установка и настройка dev модулей (prettier, eslint и т.п.)) 0,5 часа
Основные работы 8 часов
Дополнительные работы 2 часа
Список дополнительных работ: 

* Функционал удаления постов
  * Возможность удаления только своих постов
* Список людей, поставивший лайк, в отдельном popup окне
